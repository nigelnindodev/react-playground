# Counter Challenge

Create a simple counter component that displays a count and has increment (+) and decrement (-) buttons.

## Requirements

1. Display the current count value (starting at 0)
2. Add a "+" button that increments the count by 1
3. Add a "-" button that decrements the count by 1
4. The counter should not go below 0

## Hints

- Use the `useState` hook to manage the count
- Button onClick handlers should update the state
- You can use inline styles or simple CSS
